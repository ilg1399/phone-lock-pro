## 它到底怎么锁手机的

远离手机提供三种不同的锁机方式，针对不同场景各有用处：

**简单锁机**：设个倒计时，开启，完事。适合突击学习或者想要短时间专注的场景。

**番茄锁机**：用番茄工作法管理时间，锁定一段时间、休息一段时间，循环往复。25 分钟专注 + 5 分钟休息这种节奏，在时间管理圈子里被反复验证有效。你还可以自定义番茄钟时长，不一定非得是 25 分钟。
